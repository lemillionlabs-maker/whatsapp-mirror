Commands Reference

Send these from the master number in the target group:

!mirror on/off — toggle forwarding
!filter add <phone> — mute a contact
!filter remove <phone> — unmute contact
!filter list — list filtered contacts
!settings — view all settings
!set <key> <value> — change a setting
!contacts — list known contacts
!status — connection status
!help — command menu
